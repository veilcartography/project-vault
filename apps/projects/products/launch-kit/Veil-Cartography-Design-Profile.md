# Veil Cartography — Design Profile
Canonical source: Notion → "Veil Cartography — Visual Identity (draft)". This file is an export for design tools. Update Notion first, then re-export.

## Brand in one line
Mapping what has always been present but deliberately obscured — the authority of scientific illustration turned inward. Calm, precise, unhurried. The gold never shouts; it whispers.

## Two visual modes (never mix them)

### Mode A — Student Materials ("Parchment")
Workbooks, PDFs, practice guides, infographics, handouts. Warm, readable, academic.

Colours (sampled from live workbooks):
- Page background: #F4EFE9 (warm parchment)
- Panel / callout background: #EBE3D5
- Card background: #FFFFFF with 1px border #E0D2B8
- Heading ink: #4A3320
- Body text: #6A503B
- Italic subtitles / captions: #705036
- Accent (bars, rules, numbers): #8C6A4F
- Gold accent (sparingly): #B0894F, deep gold #94722F
- Hairline rules: #C9B79A

### Mode B — Social / Covers ("Deep Space")
YouTube thumbnails, social posts, course covers, promotional art.
- Background: near-black with faintest deep-violet cast (#14101A → #0D0B12)
- Primary accent: warm candlelight gold (#E9C983 highlights, #B0894F midtones)
- Only 2–3 values per composition. Extreme restraint.
- Sacred geometry (Flower of Life, Vesica Piscis, Merkaba) as spider-silk thin line work (0.5–0.8px at 1x), layered transparency, compass-and-ruling-pen precision.
- Dark ink panel for emphasis blocks in print: #33281B with text #F6EDD8 / #D8C7A6.

## Typography
- Display / headings: Cormorant Garamond (600–700). Fallback: Garamond, serif.
- Body / UI: EB Garamond (400–600). Fallback: Liberation Serif, serif.
- Eyebrow labels: small caps effect — uppercase, 11–12px, letter-spacing 0.2–0.3em, deep gold #94722F.
- Words are placed like monuments: short phrases, generous spacing, never bulk text in Mode B. Mode A allows full body text at 13–14px, line-height 1.45–1.5.
- Italic serif for subtitles, quotes, and contemplative lines.

## Layout grammar
- A4 portrait (794×1123 @96dpi) for all student PDFs; generous margins (~52px).
- Centered header stack: eyebrow → large serif title → italic subtitle → thin rule with a small gold ✦ centred on it.
- Content in bordered cards (#FBF7EF, radius 5–6px, subtle shadow) on parchment.
- Numbered steps: thin gold circle outlines (1px, #B0894F) with deep-gold numerals.
- One dark "Contemplation" panel per page maximum — the single moment of contrast.
- Sacred geometry as watermark only in Mode A: opacity ≤ 0.11, never competing with text.
- Footer: hairline rule + uppercase letterspaced line: VEIL CARTOGRAPHY · veilcartography.org · @veilcartography · © 2026

## Voice in copy
Calm, beginner-friendly, grounded. Explains, never mystifies. Second person. British English. No hype, no exclamation marks. Signature moves: "recognition, not effort", "the practice is the intention, not the image".

## Do not
- No bright/saturated colours, no gradients beyond subtle parchment/space washes.
- No sans-serif headings. No drop shadows on text. No emoji in student materials.
- Gold is never used as large fills — line work and accents only.
- Never mix Mode A and Mode B palettes in one artefact.
