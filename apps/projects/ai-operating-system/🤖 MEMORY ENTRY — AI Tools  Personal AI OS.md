
# 🤖 MEMORY ENTRY — AI Tools / Personal AI OS

## WHAT THIS PROJECT IS
A Notion-based system for recording AI tools, apps, extensions, experiments, workflows, and personal use cases. Goal: understand which tools solve which problems and build repeatable workflows from that knowledge.

## KEY SYSTEM COMPONENTS
- **AI tools database** — inventory of apps, extensions, software, AI systems tested
- **Notion email agent** — consolidates notifications, reselling alerts, content-management signals
- **AI onboarding map** — aligns collected data with strategic blueprints
- **Daily summaries** — reviews messages, subscriptions, sales notifications, security items, tasks

## TOOLS ALREADY IN USE / TRACKED
Canva, Facebook Business, TikTok Business, eBay, Website builders, Ancestry.co.uk, Gemini, Perplexity, Browser extensions, Windows apps, Mobile apps, AI agents/assistants

## DATABASE FIELDS TO BUILD
Tool Name | Type (AI assistant / browser extension / desktop / mobile / web app) | Main Use Case | Best Use | Poor Fit / Limitations | Cost / Subscription | Platform | Login / Account Used | Current Status (active / testing / paused / retired) | Related Project | Notes / Learned Knowledge | Prompt Examples | Workflow Links

## WORKFLOW ROLE CATEGORIES
Research | Capture | Summarise | Decide | Automate | Create Content | Manage Messages | Sell / List Items
