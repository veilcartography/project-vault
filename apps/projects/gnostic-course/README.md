## How this folder is structured

- `Module_01` – `Module_08` — the course, one folder per module. Each holds the written lesson (PDF), audio (MP3), and workbook where they exist.
- `00_Course_Admin` — catalogue, Gumroad listing copy, project system file. Not student-facing.
- `_Archive` — nothing live. `duplicates/` = byte-identical copies; `drafts_and_sources/` = superseded drafts + TTS scripts. Safe to delete once you're confident.
- `study/` — background research and source material (separate from the deliverable course).
