
# 🌿 MEMORY ENTRY — Amazon Volunteering / Life-Transition Gantt System

## WHAT THIS PROJECT IS
A planning and dependency-management system that organises all areas of life and work toward the larger goal of volunteering in the Amazon. An interactive Gantt chart exists and is actively used.

## PURPOSE
- Track all current goals and tasks
- Manage dependencies between projects
- Prepare the flat so returning home is easier
- Prevent loss of focus and forgotten tasks
- Translate vague thoughts into charted next steps

## KEY SYSTEM REQUIREMENTS
- Interactive Gantt chart
- Countdown to major goal
- Ability to input thoughts, unfinished tasks, and changes
- Auto-processing into: Tasks | Dependencies | Completed steps | New next actions
- Expandable view of all project areas
- Editable web page
- Links to weekly review and daily dashboard

## WHY THIS SYSTEM IS NEEDED
Unmanaged dependencies, focus lapses, memory gaps, and time-blindness can derail progress. The system should reduce cognitive load by making the next step obvious.
